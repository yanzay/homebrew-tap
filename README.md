# yanzay/homebrew-tap

Homebrew tap for [as2inspect](https://github.com/yanzay/as2inspect) — inspect
AS2 & X.509 certificates (role, expiry, fingerprint).

```sh
brew install yanzay/tap/as2inspect
```

The cask in `Casks/` is generated and updated automatically by GoReleaser on
each `as2inspect` release — do not edit it by hand.

Also available via `go install github.com/yanzay/as2inspect/cmd/as2inspect@latest`,
prebuilt binaries on the [releases page](https://github.com/yanzay/as2inspect/releases),
or in the browser at [certcutover.com](https://certcutover.com).
